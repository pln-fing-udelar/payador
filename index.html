PAYADOR

A strategy to update the fictional world-state in an Improvisational Interactive Storytelling System.

TL;DR

PAYADOR explores how Large Language Models can be grounded in structured data to increase the consistency of the world state after a user input. The system models the fictional world as components (items, characters, locations and puzzles) and uses LLMs to predict world-state Transformations that modify that state.

How it Works

PAYADOR predicts transformations to the world state in response to user input. Transformations are changes to the symbolic state of the world that are predicted by the LLM and then applied to the structured world model.

After getting the player input in free-text form, the LLM is called to suggest changes that should occur in the world, grounded in the structured representation. This ensures consistency between the predicted transformations and the symbolic state maintained by the system.
